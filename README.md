## 1. Stock Price Predictor üìà

Predicts stock prices based on historical data using Linear Regression.

**Features:**
- Downloads real-time stock data from Yahoo Finance
- Uses moving averages as features
- Visualizes actual vs predicted prices
- Predicts future stock prices

**Run:**
```bash
cd "Stock Price Predictor"
pip install -r requirements.txt
python stock_predictor.py
```

---

## 2. Twitter Sentiment Analysis üê¶

Classifies tweets as positive, negative, or neutral using NLP.

**Features:**
- Uses VADER and TextBlob for sentiment analysis
- Preprocesses tweets (removes URLs, mentions, etc.)
- Interactive mode for custom tweets
- Visualizes sentiment distribution

**Run:**
```bash
cd "Twitter Sentiment Analysis"
pip install -r requirements.txt
python sentiment_analyzer.py
```
